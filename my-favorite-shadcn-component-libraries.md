# My Favorite shadcn Component Libraries

> Source: [SyntaxPath](https://www.syntaxpath.com) — the author of SyntaxKit shares the six component libraries they keep coming back to, the same stack they reach for when building SyntaxKit.

---

## 1. Watermelon UI
* **Four libraries in one:** components, animated, blocks, and dashboard.
* **Animated section is the star:** dialogs, carousels, and tasteful micro-interactions.
* **Standard shadcn components** with small, smart modifications.
* **Blocks cover basically anything:** auth, widgets, and a hero that genuinely surprised me.
* **shadcn CLI support**, free and open source.

👉 [Visit Watermelon UI →](https://watermelon-ui.com)

---

## 2. Be UI
* **shadcn plus motion**, with one cohesive design language.
* **Focused execution:** Doesn't try to be the all-in-one library; it does one thing extremely well.
* **Huge focus** on motion and smoothness.
* **Base components** are simple but powerful; refined blocks like file upload stand out.
* **Missing for now:** more components and deeper documentation.

👉 [Visit Be UI →](https://be-ui.com)

---

## 3. Groot Studio
* **Components and templates** where the small details win.
* **A+ documentation**, especially how props are documented.
* **Strong component set** with real attention to detail.
* **Room for growth:** Could use a few more complex, opinionated components.

👉 [Visit Groot Studio →](https://grootstudio.com)

---

## 4. Vengeance UI
* **A deep catalog** of components plus portfolio templates.
* **Tons to choose from:** candy button, liquid metal, morph text, book, bento grid, staggered grid, cursor card.
* **High code quality:** Huge fan of the code quality, it ships with actual comments.
* **Constructive note:** Would love a clearer, more unified design language.

👉 [Visit Vengeance UI →](https://vengeance-ui.com)

---

## 5. Kairo UI
* **Clean, modern primitives** for shipping fast.
* **A focused set** of polished components that drop straight into a shadcn project.
* **Sensible defaults** that look good with zero tweaking.

👉 [Visit Kairo UI →](https://kairo-ui.com)

---

## 6. Dot Matrix
* **A distinctive take on UI** with a strong visual identity.
* **Real character:** Worth a look for components with real character, not just another shadcn reskin.
* **Great for adding personality** to an otherwise standard stack.

👉 [Visit Dot Matrix →](https://dotmatrix.com)
